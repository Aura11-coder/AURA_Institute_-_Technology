# 🌐 Aura Institute & Technology Website

A modern and responsive educational website developed for **Aura Institute & Technology**. The website provides information about courses, internships, placements, scholarships, student reviews, and online examinations.

---

## 🚀 Features

- Responsive Design
- Modern UI/UX
- Course Showcase
- Placement Information
- Student Reviews
- Gallery Section
- SEO Optimized
- Fast Loading
- Mobile Friendly

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
